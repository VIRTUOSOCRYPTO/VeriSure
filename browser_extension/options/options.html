API Configuration

Configure the VeriSure API endpoint. By default, it connects to your local instance. If you're using a deployed instance, update the URL below.

Example: https://your-verisure-app.com or http://localhost:8001

Extension Features

  • Right-click context menu for selected text
  • Right-click context menu for images
  • Quick analysis popup interface
  • Full page text scanning
  • Inline result display on web pages
  • Analysis history (last 50 items)
  • Browser notifications
  • Risk level badges

How to Use

Quick Analysis:
Click the VeriSure icon in your browser toolbar to analyze any text or URL.

Context Menu:
Right-click on selected text or images and choose "Analyze with VeriSure".

Inline Analysis:
Select text on any webpage and click the floating "Analyze" button that appears.
